# Icons

UI icons are from [Heroicons](https://heroicons.com/) v2 (MIT License), by the makers of Tailwind CSS.

- Source package: `heroicons` (outline 24×24)
- Files in this folder are vendored SVGs used via CSS `mask-image` in `styles.css`
- App / tray assets under `electron/assets/` are rasterized from Heroicons paths

See the root `NOTICE` file for the full license text.
