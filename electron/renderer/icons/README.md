# Icons

UI icons are from [Heroicons](https://heroicons.com/) v2 (MIT License), by the makers of Tailwind CSS.

- Source package: `heroicons` (outline 24×24)
- Files in this folder are vendored SVGs used via CSS `mask-image` in `styles.css`
- The menu bar template under `electron/assets/` uses the supplied Typephone status-bar artwork
- The application icon under `electron/assets/` uses the dedicated Typephone macOS artwork
- Sidebar branding under `electron/renderer/assets/` uses the supplied Typephone logo and wordmark SVGs

See the root `NOTICE` file for the full license text.
